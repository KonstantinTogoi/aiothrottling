# aiothrottling

aiothrottling is a library with throttling utilities that can be used with asyncio.

## Installation

```python
pip install aiothrottling
```

or

```python
python setup.py install
```

## Supported Python Versions

Python 3.5, 3.6, 3.7 and 3.8 are supported.
